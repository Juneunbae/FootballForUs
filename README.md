# FootballForUs
축구 커뮤니티

```
축구에 대한 얘기를 할 수 있는 축구 커뮤니티 사이트입니다.
만든 이와 다른 다양한 축구 의견을 들으려 만들어졌습니다.
또, 가입 등급에 따라 글을 쓰거나 읽는 권한이 없어 누구나 회원가입과 로그인을 하시면, 자유롭게 소통할 수 있습니다.
```
